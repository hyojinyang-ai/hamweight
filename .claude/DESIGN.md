# MyWieght DESIGN.md

## Style Direction
Paper.design-inspired product interface adapted for a mobile wellness product.

The UI should feel:
- editorial and product-focused
- monochrome first with restrained accents
- clean, airy, and precise
- minimal rather than ornamental

## Principles
- Typography carries the brand more than color
- Surfaces should feel clean and flat with subtle structure
- Use soft neutral fills and thin dark borders
- Avoid visual noise and unnecessary decoration

## Typography
- Primary UI font: Geist Sans
- Tight tracking on headings
- Medium/light body text
- Bold only for key product labels and metrics

## Components
- Buttons: simple rounded pills, black primary, pale secondary
- Inputs/selects: soft neutral capsules, clear contrast
- Cards: light surfaces with thin borders and faint shadows
- Navigation: clean floating dock, monochrome
- Sheets/modals: solid, quiet surfaces with generous spacing

## Motion
- Subtle opacity/transform only
- No exaggerated bounce
- Product-like transitions
